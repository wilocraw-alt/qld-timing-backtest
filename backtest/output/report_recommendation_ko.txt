전략 추천 — 패널 확률 × (강세장 수익 + 닷컴 위기) 기대값 분석
================================================================

※ 본 분석은 우리 백테스트와 패널 토의 확률을 결합한 것이며 투자 조언이 아님.
  시나리오 수익은 과거 두 사례에 근거한 추정치(미래 보장 아님).
  재현: python3 backtest/recommendation.py

■ 입력 1 — 패널 종합 확률 (12~24개월)
  강세 지속        : 42%
  의미 있는 조정   : 38%   (-15~35% 후 부분회복)
  닷컴급 폭락      : 11%   (-50%+ 장기)
  스태그/침체      :  9%
  → 비(非)강세 합계 58% > 강세 42%

■ 입력 2 — 시나리오별 18개월 수익 가정의 근거
  강세 = 2011~2026 백테스트 MWRR을 18개월 복리환산
         (QLD 33.5%/yr, TQQQ 42.8%/yr, 추세/레짐 ~25%/yr)
  폭락 = 닷컴 시나리오(1x -15% / 3x -97% / 추세추종 +147~339%)를 2x·보수적 매핑
  조정/스태그 = 두 사례 사이값 + 레버리지 변동성손실(decay) 반영

■ 기대값 결과 (높은 순)

  ◆ QLD + 추세이탈(200MA)
    기대수익   : +14.5%   (1위)
    최악(폭락) : +10%
    강세 상단  : +38%

  ◆ 3x + 추세이탈(200MA)
    기대수익   : +12.8%
    최악(폭락) : +5%
    강세 상단  : +41%

  ◆ 바벨 (60% QLD보유 + 40% 3x추세)
    기대수익   : +12.0%
    최악(폭락) : -31%
    강세 상단  : +49%

  ◆ QLD 적립 (2x 무방비 보유)
    기대수익   : +11.5%
    최악(폭락) : -55%
    강세 상단  : +54%

  ◆ 3x 무방비 보유 (TQQQ/SOXL)
    기대수익   : -1.0%   (꼴찌)
    최악(폭락) : -88%
    강세 상단  : +71%   (상단만 최고)

■ 핵심 통찰
  1) 무방비 3x는 지금 국면에서 '최악'으로 뒤집혔다.
     순수 강세장 백테스트 1위였으나, 비강세 확률 58%에서 기대수익 -1%.
     강세 상단(+71%)을 폭락(-88%)·조정·횡보 decay가 통째로 상쇄.
  2) 추세이탈(200일선) 규칙이 기대수익을 끌어올린다.
     강세 상단 일부 포기(+71%→+41%) 대신 폭락에서 생존(+5~10%) → 기대값 상위.
     닷컴 백테스트의 '폭락장에선 타이밍이 구원자'가 확률가중에서도 성립.
  3) 레버리지엔 추세규칙이 필수: 같은 3x라도 추세이탈 부착 시 -1% → +12.8%.

■ 추천 (위험성향별)
  ● 기본(균형) — '추세관리형 바벨'
    - 핵심 60%: QLD(2x) 월적립(DCA) 보유 — 생존 가능한 베이스, 강세 포착
    - 위성 40%: TQQQ/SOXL(3x)에 200일선 이탈 시 현금화 규칙 '필수'
                (TQQQ·SOXL 분산 = 앞서 Sharpe 0.67로 개선 확인)
    - 집행: 월 분할(DCA) — 닷컴 테스트에서 분할이 레버리지 재앙 완충 확인
    - 기대 +12%, 폭락 -31% (무방비 2x -55% / 3x -88%보다 방어적)
  ● 규칙 실행이 어렵다(매도신호 못지킴) → QLD 월적립. 기대 +11.5%, 폭락 -55% 감수.
  ● 최대 방어 + 양호한 기대 → QLD+추세이탈 (기대 1위 +14.5%, 폭락에도 +10%).
  ● 공격적이되 전멸 회피 → 3x+추세이탈 (+12.8%, 폭락 +5%).
  ● 무방비 3x 장기보유 → 비추천 (현재 확률에서 기대값 마이너스).

■ 민감도 (정직한 한계)
  - 결론은 '강세 확률'에 민감하다:
      강세 42%(패널)  → 1위 QLD+추세이탈
      강세 55~65%     → 1위 QLD적립(2x 보유)
      강세 75%+       → 비로소 3x 무방비보유가 1위
    즉 무방비 3x가 정당화되려면 '강세 75% 이상' 확신이 필요. 패널 합의(42%)와 거리가 큼.
  - 추세추종은 횡보·휩쏘장에서 가장 취약(스태그 시나리오에서 마이너스).
  - 시나리오 수익은 과거 2사례 기반 추정 — 미래 폭락은 양상이 다를 수 있음.

■ 한 줄 결론
  "강세에 올인"하지 말고 "틀려도 살아남게" 설계하라.
  데이터·확률 모두 — 레버리지는 쓰되 추세이탈 규칙을 붙인 바벨이 현재 국면의 최적해.
  무방비 3x 장기보유는 지금 기대값이 마이너스다.
