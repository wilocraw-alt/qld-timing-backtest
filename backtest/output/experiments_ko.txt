=========================================================
QLD 전략 추가 실험 종합 (트랜치/익절/일한도)
=========================================================

대상/기간/자본: QLD, 2006-06-21 ~ 2026-06-18, 초기 $10,000
기준전략 파라미터: n=5,m=50,a=10,b=15,c=20,d=40,e=10,f=20

---------------------------------------------------------
실험 1: 트랜치 사이징 스윕 (일한도 $1,000 유지)
---------------------------------------------------------
결과: ts=1000 / 2500 / 5000 가 모두 동일 (최종 $34,052, CAGR 6.32%, MDD 25.7%)
해석: 일 매수 한도 $1,000이 실질 제약. 큰 트랜치도 여러 날에 쪼개 체결되어
      같은 금액이 같은 가격대에 투입 → 트랜치 크기 무의미.
결론: 실효 레버는 트랜치 크기가 아니라 '일 매수 한도(배치 속도/규모)'.

---------------------------------------------------------
실험 2: 익절 임계 상향 (c=20%/d=40% → c=100%/d=200%)
---------------------------------------------------------
기준(c20/d40): 최종 $34,052 / CAGR 6.32% / MDD 25.7% / 현금 81% / 거래 290
상향(c100/d200): 최종 $71,235 / CAGR 10.32% / MDD 42.5% / 현금 56% / 거래 209
해석: 익절을 늦추면 더 오래 보유 → 수익 2.09배, 단 낙폭도 크게 확대.
      (사실상 익절을 거의 끄는 no-takeprofit($75,253)에 근접)

---------------------------------------------------------
실험 3: 일 매수 한도 제거 (full 전략)
---------------------------------------------------------
한도 $1,000: 최종 $34,052 / CAGR 6.32% / MDD 25.73% / 현금 81.05%
한도 무제한: 최종 $34,192 / CAGR 6.34% / MDD 25.30% / 현금 81.25%
해석: 한도 제거만으로는 거의 변화 없음(+0.4%). 한도가 실제로 걸린 경우가 드묾
      (대부분 현금 보유 + 같은 날 다중신호 동시발동 희소).

---------------------------------------------------------
실험 4: 일한도 무제한 + 트랜치 사이징 스윕  ★핵심
---------------------------------------------------------
ts=1000: 최종 $34,192  / CAGR 6.34%  / MDD 25.3% / 현금 81%
ts=2500: 최종 $68,810  / CAGR 10.13% / MDD 46.5% / 현금 69%
ts=5000: 최종 $125,033 / CAGR 13.47% / MDD 49.0% / 현금 61%
해석: 한도를 없애면 트랜치 크기가 강하게 작동. 1회 신호당 더 많이 배치할수록
      수익 단조 증가(현금비중 81→61% 동반), 단 MDD도 25→49%로 확대.
      거래수는 290→282로 거의 불변(신호 패턴 동일, 1회 집행규모만 변화).

=========================================================
종합 결론
=========================================================
1) 본 전략의 저조(기준 $34k vs lumpsum $996k)의 진짜 원인은 '일 매수 한도'가 아니라
   (a) 적은 배치 규모(현금 ~81% 유휴)와 (b) 익절·하락장매도가 99배 상승장을 잘라낸 것.
2) 수익을 키우는 레버 순위: 배치 규모↑(트랜치/한도) > 익절 완화 > 하락장매도 완화.
   단 모두 MDD를 크게 키운다(수익-낙폭 trade-off).
3) 전략의 본질적 장점은 낮은 MDD(현금 보유)에 있으나, 레버리지 ETF의 장기 추세장에서는
   buy&hold 대비 총수익·위험조정수익 모두 열위.
