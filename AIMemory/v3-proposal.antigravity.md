## 구조적 전제 인식

롱/현금 전략은 최대 노출 100%를 넘을 수 없기 때문에, QLD처럼 강하게 우상향하는 자산에서 buy&hold를 이기려면 **행동 공간을 확장**해야 합니다. 아래 5개 후보는 각자 다른 방식으로 이 한계를 돌파합니다.

---

## 후보 1: QQQ 모멘텀 시그널 → QLD 레버리지 타이밍 (레버리지 ETF 변동성 손실 회피형)

### 1. 메커니즘
- **신호**: 비레버리지 기초지수인 QQQ의 MA/모멘텀(12개월 수익률, 200일 이동평균 대비)을 관찰.
- **행동**: QQQ가 상승 레짐이면 QLD(2x)를 100% 보유. QQQ 추세 붕괴 시 QLD 청산 후 현금화. 반등 확인 시 재진입.
- **레버리지 ETF decay 회피**: QLD는 횡보·하락 구간에서 변동성 손실이 2배로 누적되므로, 이 구간만 피하면 단순 보유 대비 복리 기반이 높아집니다.

### 2. 왜 구조적으로 이길 수 있나
- QLD 자체를 신호로 쓰면 가격에 이미 반응한 신호다. **QQQ를 선행 지표**로 쓰면 레버리지 ETF의 변동성 손실이 가장 크게 발생하는 "횡보+하락" 구간을 2x 레버리지 없이 버틸 수 있음.
- 2022년처럼 QQQ가 -30% 하락하면 QLD는 -55%+변동성손실로 훨씬 더 내려감 → 이 구간 현금 보유 시 복리 기반 보존 효과가 2x.

### 3. 필요한 엔진 확장
- `loader.py`로 QQQ 데이터 추가 로드 (이미 가능).
- `indicators.py`에 QQQ 기반 신호 컬럼 추가.
- 현재 롱+현금 엔진으로 충분 (숏/마진 불필요).

### 4. 오버피팅 위험 + 검증법
- **위험**: MA 윈도우 선택(200일)이 QLD 역사에 과적합될 수 있음.
- **검증**: 파라미터 라운드넘버 사용, 2006~2011(학습) / 2011~2026(검증)으로 out-of-sample 분리. 다른 레버리지 ETF(TQQQ, UPRO)로 전이 테스트.

### 5. 현실성
- 매우 높음. QQQ는 유동성 최상위, 추가 수수료 거의 없음. 실행 복잡도 낮음.
- **한계**: 2020년 3월처럼 V자 반등 시 재진입 지연. 2017~2021처럼 장기 강세장에서는 buy&hold와 성과 동일.

## 후보 2: 변동성 타겟팅(Vol-Targeting) + 레버리지 스케일링

### 1. 메커니즘
- **신호**: QLD의 20일 실현 변동성(σ₂₀). 저변동성 구간 → 레버리지 확대, 고변동성 구간 → 레버리지 축소.
- **행동**: 목표 연환산 변동성 σ_target=15%로 설정. 매일 노출 비중 = σ_target / σ₂₀. 상한 150%, 하한 50% 클리핑.
  - σ₂₀ 낮을 때(예: 10%) → 노출 150% (마진/TQQQ 회전 필요)
  - σ₂₀ 높을 때(예: 40%) → 노출 37.5% (현금 다수)

### 2. 왜 구조적으로 이길 수 있나
- **노출 > 100%**: 저변동성 강세장에서 레버리지를 150%까지 키우므로 buy&hold(100%) 초과 수익 가능.
- **하락 방어**: 고변동성(= 주로 하락장)에서 자동 노출 축소. 2020 코로나, 2022 금리인상 구간에서 기계적으로 방어.
- 학술적으로 "변동성은 군집(clustering)"하므로 전일 변동성이 내일 변동성의 좋은 예측자임.

### 3. 필요한 엔진 확장
- 노출 비중을 소수점(150%, 75% 등)으로 표현하는 **포지션 사이징** 모듈 필요.
- 150% 이상은 마진 또는 TQQQ(3x) 일부 혼합으로 구현 → 추가 티커 지원 필요.
- 마진 미사용 시 상한을 100%로 제한하면 단순히 "저변동 = 풀투자, 고변동 = 현금"으로 단순화 가능 (롱+현금 엔진 유지).

### 4. 오버피팅 위험 + 검증법
- **위험**: σ_target, 창 크기(20일) 조합에 따라 결과가 크게 달라질 수 있음.
- **검증**: 파라미터 민감도 분석(σ_target=10~20%, 창=10~30일). 다른 자산(TQQQ, SPY)에서도 동일 파라미터로 수익 확인.

### 5. 현실성
- 노출 ≤ 100%로 제한 시 마진 불필요, 높은 실행 가능성. 일별 리밸런싱 거래비용 발생(연간 수백 회).
- **한계**: 거래비용이 수익을 잠식할 수 있음. QLD 특유의 높은 변동성으로 리밸런싱 빈도 높음.

## 후보 3: QLD + SQQQ(인버스 3x) 롱/숏 페어 (순수 레짐 스위칭)

### 1. 메커니즘
- 상승 레짐: QLD 100% 보유.
- 하락 레짐: QLD 청산 + SQQQ(나스닥 -3x) 25~33% 매수 → 하락 구간에서 절대 수익 창출.
- 레짐 판정: MA200 기반 (가장 단순, 파라미터 1개).

### 2. 왜 구조적으로 이길 수 있나
- 하락장에서 현금이 아니라 **SQQQ 수익**이 발생하므로 100% 롱 대비 절대적으로 유리.
- buy&hold가 -50% 하락할 때 SQQQ 보유분이 +수익을 내므로 전체 포트폴리오는 손실 최소화.
- 단, 하락 레짐 진입 직후 급반등 시 SQQQ 포지션이 손실로 전환 위험.

### 3. 필요한 엔진 확장
- **숏/인버스 포지션 지원**: `portfolio.py`에 `short_shares`, `inverse_ticker` 개념 추가 필요.
- SQQQ 가격 데이터 추가 로드.
- 인버스 ETF는 시간이 지날수록 변동성 손실로 가치 감소 → **보유 기간 제한** 로직 필요.

### 4. 오버피팅 위험 + 검증법
- **위험**: 레짐 판정 기준(MA200)이 QLD 하락 타이밍과 완벽히 맞지 않을 때 SQQQ 포지션이 큰 손실.
- **검증**: 레짐 판정 오류 시나리오(V자 반등) 스트레스 테스트. SQQQ 비중(25%/33%/50%) 민감도 분석.

### 5. 현실성
- SQQQ는 실존 ETF, 유동성 충분.
- **한계**: 인버스 ETF 자체의 변동성 손실(decay)이 심각 → 장기 보유 불가, 반드시 단기 포지션만. 2x+3x 인버스 결합으로 복잡성 상승.
- **솔직한 평가**: 레짐 판정이 틀릴 때의 손실 리스크가 크고, SQQQ의 decay로 실제 수익이 이론보다 낮음.

## 후보 4: 크로스에셋 모멘텀 로테이션 (QLD/TLT/현금 3-way)

### 1. 메커니즘
- 매월 말 3개 자산(QLD, TLT[장기 국채], 현금)의 12개월 모멘텀 점수 비교.
- 가장 높은 모멘텀 자산 1개에 100% 투자. 또는 상위 2개에 50:50.
- **QLD 강세** → QLD 100%, **채권 강세(주식 약세)** → TLT 100%, **둘 다 약세** → 현금.

### 2. 왜 구조적으로 이길 수 있나
- TLT로 전환 시 **주식 하락 방어 + 채권 수익** 동시 달성 (2022년 예외지만 역사적으론 음의 상관).
- 주식 하락 구간의 현금/채권 수익이 복리 기반을 보존.
- QLD 강세 구간에서는 100% QLD로 buy&hold와 동일한 노출 달성.

### 3. 필요한 엔진 확장
- TLT 데이터 로드 및 포트폴리오 전환 로직.
- 월간 리밸런싱 타이밍 추가.

### 4. 오버피팅 위험 + 검증법
- **위험**: 2022년처럼 주식+채권 동반 하락 시 완전 실패. 모멘텀 팩터 자체가 crowded trade화.
- **검증**: 다른 시장(유럽, EM)에서 동일 전략 전이. 회전 자산 수/종류 변화에 따른 민감도 분석.

### 5. 현실성
- 월 1회 거래 → 거래비용 최소. TLT는 유동성 충분.
- **한계**: 2022년 금리인상 환경에서 주식+채권 동반 하락으로 이 전략은 완전 실패. 고금리 환경에서는 현금만이 대안.

## 후보 5: 평균회귀 스윙 트레이딩 (RSI 기반 단기 역추세)

### 1. 메커니즘
- **신호**: QLD의 5일 RSI(과매도 < 25, 과매수 > 75).
- **행동**: RSI < 25 → 매수(단기 반등 기대), RSI > 75 → 매도(단기 되돌림 기대).
- **배경**: 레버리지 ETF는 단기 과매도 후 강한 기술적 반등이 자주 발생.
- 포지션: 신호당 자본의 20~30% 진출입, 최대 80% 노출.

### 2. 왜 구조적으로 이길 수 있나
- 단기 평균회귀는 롱/현금 한계 내에서도 buy&hold 초과 가능.
- 과매도 진입 → 반등 수익 → 과매수 익절의 반복으로 복리 기반 지속 성장.
- **단, QLD는 강한 추세 자산이므로** 추세 방향에 맞는 거래만 선택(상승 레짐 내 과매도만 매수).

### 3. 필요한 엔진 확장
- RSI 계산 추가(rolling, 과거 데이터만, 미래참조 없음).
- 부분 포지션 사이징 지원 (현재 엔진 수정 최소).

### 4. 오버피팅 위험 + 검증법
- **위험**: RSI 임계값(25/75)과 포지션 비중 조합에 심각한 과적합 위험.
- **검증**: in-sample(2011~2018) / out-of-sample(2018~2026) 분리. 파라미터 ±5 범위 민감도 분석. 다른 자산에서 동일 파라미터 검증.
- **솔직한 평가**: 단기 트레이딩은 거래비용과 슬리피지에 민감하며, 백테스트 과적합 위험이 가장 높은 분류. 실제 성과는 백테스트보다 크게 낮을 가능성이 큼.
