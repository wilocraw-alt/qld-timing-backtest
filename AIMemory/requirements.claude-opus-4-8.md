## 1. 목표
QLD(ProShares Ultra QQQ, 2x 레버리지 ETF)를 대상으로, 규칙기반 투자시점(매수/매도) 전략을
백테스트하고 **각 규칙의 ablation(on/off) 효과**와 **두 비교군 대비 성과**를 검증한다.

## 2. 대상·데이터·기간·자본
- 종목: **QLD** (필요 시 데이터 검증용 보조: QQQ)
- 데이터원: yfinance (조정종가 기준). 인터넷 불가 시 캐시/대체원 명시.
- 기간: **QLD 전체기간** (상장 ~2006-06 ~ 현재)
- 초기자본: **$10,000**

## 3. 신호 전략 규칙 (초기 파라미터)
n=5, m=50, a=10, b=15, c=20, d=40, e=10, f=20

1. **대형 하락장 판단/축소**: 종가가 200일 이동평균선을 **n=5일 연속 하회**하면 대형 하락장으로
   판단 → 현재 **보유 주식 수의 m=50% 매도**.
2. **ATH(전고점) 기준 분할매수**: ATH 대비 **-a=10%** → 1차 매수, **-b=15%** → 2차 매수.
3. **전저점 기준 분할매수**: 전저점 대비 **+e=10%** → 3차 매수, **+f=20%** → 4차 매수.
4. **익절(분할매도)**: 매수 평단가 대비 **+c=20%** → 1차 매도, **+d=40%** → 2차 매도.

## 4. 포지션 사이징 / 자금 운용 (확정)
- **매수 회차당 = 총자본의 균등 10% = $1,000** (1~4차 각 $1,000). 현금 보유가 목적이므로
  신호로 배치되는 최대 노출은 약 40%, 나머지는 현금. (정확한 현금/재투자 규칙은 설계서 확정)
- **일 매수 상한 = 고정 $1,000/일.** 어떤 날의 목표 매수액이 상한을 넘으면 **다음 날로 이월**,
  목표 분할매수 가격 조건이 유지되는 동안 여러 날에 걸쳐 나누어 매수.
- 매도(익절·하락장 축소)는 주식 수 기준 비율 매도.

## 5. 전고점·전저점 판단 (설계서에서 표준기법 제안·확정)
- **요구**: "충분한 구간을 두고" 판단. 매일 출렁일 때마다 전고/전저를 갱신하면 전략이 깨짐.
- 표준 후보: (a) rolling window 극값(예: 좌우 N일 swing high/low = fractal pivot),
  (b) ZigZag(임계 % 반전), (c) 확정 지연(confirmed pivot, lookback+lookforward).
- 설계서에서 표준기법을 제안하고 윈도우/임계값 파라미터를 명시. 없으면 아이디어로 처리.

## 6. Ablation (규칙 on/off 효과 검증)
각 규칙 컴포넌트를 끄고/켜 효과 측정. 최소 변형:
- Full(전체) / no-200MA(하락장 축소 제거) / no-ATH-buy / no-prevlow-buy /
  no-takeprofit(익절 제거) / 개별 단일규칙 등. (설계서에서 ablation 매트릭스 확정)

## 7. 비교군 (baselines)
- **A. 초기 일괄투자**: 1일차 $10,000 전액 QLD 매수 후 buy&hold.
- **B. 1/100 분할매수**: $10,000을 100등분($100), **전체기간 균등분산**(예: 동일 간격/매월)
  매수 후 보유. 신호전략과 동일 기간 배정 비교.
